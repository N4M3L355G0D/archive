wordlist2db-createdb.py -- create the database specified in config
wordlist2db-createtable.py -- create the table wordlist in the database specified in config
wordlist2db-insert.py -- insert entries and id's into database specified in config using a wordlist provided as the only argument

config consists of four lines, each with a configuration option; they are as below.

for logging into the sql server
user=
password=
address=
for the database operations
database=
